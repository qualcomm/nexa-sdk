## **IoT Devices**

Supported Qualcomm IoT platforms for edge AI inference in smart cameras, robotics, industrial automation, and embedded systems.

### Supported Chipsets
- Qualcomm QCS9075
- Qualcomm QCS9100
- Qualcomm QCS8275

### Get Started

**Option 1: Physical Device**

Use a Qualcomm IoT development kit (e.g., Dragonwing RB3 Gen 2 or QCS8550 HDK). Install GenieX on the Linux Arm64 image and run inference.


**Option 2: Qualcomm Developer Cloud**

Click "IOT" at the [Qualcomm Developer Cloud - IOT page](https://qdc.qualcomm.com/devices/iot) and open a `Qualcomm Dragonwing IQ-9075` or `Qualcomm Dragonwing IQ-9100 Beta` device.

<Tip>
**SSH key permission error?** If you see `WARNING: UNPROTECTED PRIVATE KEY FILE!` when connecting, your `.pem` file permissions are too open. Fix with:

```bash
chmod 600 /path/to/your-key.pem
```

SSH requires private keys to be readable only by the owner (mode `600`).
</Tip>
