GenieX runs on Snapdragon across three operating systems. Pick the row that matches where you want to run.

| OS | Chipsets | Interfaces |
|---|---|---|
| **Windows ARM64** | <ul><li>Snapdragon X Elite</li><li>Snapdragon X2 Elite</li></ul> | <ul><li>[CLI](/en/run/cli/quickstart)</li><li>[Python SDK](/en/run/python/quickstart)</li><li>[Local Server](/en/run/cli/local-server)</li></ul> |
| **Android** | <ul><li>Snapdragon 8 Elite (`SM8750`)</li><li>Snapdragon 8 Elite Gen 5 (`SM8850`)</li></ul> | <ul><li>[Android SDK](/en/run/android/quickstart) (Kotlin, Maven Central)</li></ul> |
| **Linux ARM64** | <ul><li>Qualcomm QCS9075</li></ul> | <ul><li>[Native install](/en/run/cli/install)</li><li>[Docker](/en/run/linux/install)</li></ul> |

## **No physical device on hand?**

Sign in to [Qualcomm Developer Cloud (QDC)](https://qdc.qualcomm.com/) for remote sessions on Snapdragon chipsets — Windows ARM64, Android, and Linux ARM64 are all available. See the [QDC walkthrough in the FAQ](/en/resources/faq#qdc-qualcomm-device-cloud) for setup and SSH instructions.
