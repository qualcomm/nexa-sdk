## **Before you start**

- **Hardware** — GenieX targets Snapdragon. See [Supported devices](/en/get-started/devices) for compute, mobile, and IoT chipsets.
- **Backend** — your model dictates which runtime path you take. See [Choose your backend](/en/get-started/choose-backend).
- **Models** — see [Supported models](/en/models/supported) for tested examples on each backend.
