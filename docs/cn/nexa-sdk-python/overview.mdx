## **Nexa Python SDK**
NexaAI 提供覆盖多模态的本地 AI 推理 Python API。
支持的大类包括：LLM、VLM、文本向量嵌入、文档重排、计算机视觉（OCR）、自动语音识别（ASR）、文本转语音（TTS）、图像生成、说话人分离（Diarize）、KV 缓存、采样配置、聊天模板与错误处理等。
