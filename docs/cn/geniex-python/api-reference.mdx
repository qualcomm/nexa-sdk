# GenieX Python SDK API 参考

GenieX Python SDK 遵循与 Hugging Face `transformers` 相同的设计模式——使用 `AutoModel*.from_pretrained()` 加载模型，然后调用 `.generate()` 进行推理。

## AutoModelForCausalLM

用于加载因果语言模型的工厂类——同时支持纯文本和多模态模型。对于纯文本模型返回 [`GeniexLLM`](#geniexllm)，检测到多模态模型（如 phi4_multimodal、qwen3.5-vl、gemma4）时返回 [`GeniexVLM`](#geniexvlm)。

```python
from geniex import AutoModelForCausalLM

model = AutoModelForCausalLM.from_pretrained(
    "Qualcomm/Qwen3-4B-Instruct",   # HuggingFace 仓库 ID、简称别名或本地路径
    device_map="auto",               # "auto" | "<plugin>" | "<plugin>:<device>"
)
```

### `from_pretrained()`

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `model_name_or_path` | `str` | *必填* | HuggingFace 仓库 ID（如 `"Qualcomm/Qwen3-4B-Instruct"`）、简称别名（如 `"qwen3"`）或本地路径。 |
| `model_name` | `str \| None` | `None` | 覆盖注册表模型名称（如 QAIRT 的 `"granite4"`）。默认为 `model_name_or_path`。 |
| `quant` | `str \| None` | `None` | 量化变体（如 `"Q4_K_M"`）。用于从 HuggingFace Hub 下载时过滤文件。 |
| `device_map` | `str` | `"auto"` | `"auto"` 自动选择第一个可用插件/设备。也可指定 `"<plugin_id>"` 或 `"<plugin_id>:<device_id>"`。 |
| `n_ctx` | `int` | `0` | 上下文长度（`0` = 模型默认值）。 |
| `n_gpu_layers` | `int` | `-1` | 卸载到 GPU 的层数（`-1` = 全部）。 |
| `tokenizer_path` | `str \| None` | `None` | 覆盖分词器文件路径。 |
| `license_id` | `str \| None` | `None` | NPU 许可证 ID。 |
| `license_key` | `str \| None` | `None` | NPU 许可证密钥。 |
| `mmproj_path` | `str \| None` | `None` | 多模态投影器文件路径。从 Hub 下载时自动解析；显式传入可强制启用 VLM 模式。 |
| `hf_token` | `str \| None` | `None` | HuggingFace 令牌，用于访问受限模型。 |
| `enable_thinking` | `bool` | `False` | 启用思考模式（如 Qwen 推理模型）。 |
| `n_threads` | `int` | — | 推理线程数。 |
| `n_batch` | `int` | — | 提示处理的批大小。 |
| `max_tokens` | `int` | — | 生成的最大 token 数。 |
| `system_prompt` | `str` | — | 预置系统提示。 |

**返回值：** [`GeniexLLM`](#geniexllm) 或 [`GeniexVLM`](#geniexvlm)（根据模型自动检测）

## AutoModelForVision2Seq

用于加载视觉语言/多模态模型的工厂类。返回 [`GeniexVLM`](#geniexvlm) 实例。

```python
from geniex import AutoModelForVision2Seq

model = AutoModelForVision2Seq.from_pretrained("qwen3vl", device_map="auto")
```

### `from_pretrained()`

接受与 [`AutoModelForCausalLM.from_pretrained()`](#from_pretrained) 相同的所有参数，额外支持：

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `mmproj_path` | `str \| None` | `None` | 多模态投影器文件路径。从 Hub 下载时自动解析。 |

**返回值：** [`GeniexVLM`](#geniexvlm)

## GeniexLLM

由 `AutoModelForCausalLM.from_pretrained()` 返回的纯文本语言模型实例。

### `generate()`

从格式化的提示字符串运行文本生成。

```python
output = model.generate(prompt, max_new_tokens=256)
print(output.text)
```

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `prompt` | `str` | *必填* | 格式化的提示字符串（使用 `model.tokenizer.apply_chat_template()` 构建）。 |
| `max_new_tokens` | `int` | `512` | 生成的最大 token 数。 |
| `temperature` | `float` | `0.7` | 采样温度。 |
| `top_p` | `float` | `0.9` | 核采样阈值。 |
| `top_k` | `int` | `40` | Top-k 采样。 |
| `min_p` | `float` | `0.0` | 最小概率阈值。 |
| `repetition_penalty` | `float` | `1.1` | 重复惩罚。 |
| `presence_penalty` | `float` | `0.0` | 存在惩罚。 |
| `frequency_penalty` | `float` | `0.0` | 频率惩罚。 |
| `seed` | `int` | `-1` | 随机种子（`-1` = 随机）。 |
| `stop` | `list[str] \| None` | `None` | 停止序列。 |
| `grammar` | `str \| None` | `None` | GBNF 语法字符串，用于约束生成。 |
| `json_mode` | `bool` | `False` | 强制 JSON 输出。 |
| `stream` | `bool` | `False` | 设为 `True` 则返回 [`TextIteratorStreamer`](#textiteratorstreamer)。 |

**返回值：** [`GenerateOutput`](#generateoutput)（`stream=True` 时返回 [`TextIteratorStreamer`](#textiteratorstreamer)）

### `reset()`

重置对话状态并清除 KV 缓存。

### `save_kv_cache(path)`

将当前键值缓存保存到文件。

| 参数 | 类型 | 说明 |
|---|---|---|
| `path` | `str` | 保存缓存的文件路径。 |

### `load_kv_cache(path)`

从文件加载之前保存的键值缓存。

| 参数 | 类型 | 说明 |
|---|---|---|
| `path` | `str` | 加载缓存的文件路径。 |

### `close()`

释放模型句柄并释放资源。也支持上下文管理器用法：

```python
with AutoModelForCausalLM.from_pretrained("qwen3") as model:
    output = model.generate(prompt)
```

## GeniexVLM

由 `AutoModelForVision2Seq.from_pretrained()` 返回的视觉语言模型实例。

### `generate()`

与 [`GeniexLLM.generate()`](#generate) 参数相同，额外支持：

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `images` | `list[str] \| None` | `None` | 模型处理的图像文件路径列表。 |
| `audios` | `list[str] \| None` | `None` | 模型处理的音频文件路径列表。 |

```python
output = model.generate(prompt, images=["/path/to/image.jpg"], max_new_tokens=256)
print(output.text)
```

**返回值：** [`GenerateOutput`](#generateoutput)（`stream=True` 时返回 [`TextIteratorStreamer`](#textiteratorstreamer)）

### `reset()`

重置对话状态并清除 KV 缓存。

### `close()`

释放模型句柄并释放资源。支持上下文管理器用法。

## ModelTokenizer

通过 `model.tokenizer` 访问。提供兼容 `transformers` 的聊天模板接口。

### `apply_chat_template()`

使用模型内置聊天模板格式化消息列表。

```python
messages = [{"role": "user", "content": "What is 2+2?"}]
prompt = model.tokenizer.apply_chat_template(
    messages, tokenize=False, add_generation_prompt=True,
)
```

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `messages` | `list[dict]` | *必填* | 包含 `"role"` 和 `"content"` 键的消息字典列表。 |
| `tokenize` | `bool` | `False` | 必须为 `False`——不支持独立分词。 |
| `add_generation_prompt` | `bool` | `True` | 是否追加生成提示后缀。 |
| `enable_thinking` | `bool` | `False` | 启用思考模式（如 Qwen 推理模型）。 |
| `tools` | `list[dict] \| str \| None` | `None` | 工具定义，可为字典列表或预序列化 JSON 字符串。 |

**返回值：** `str`——格式化后的提示，可直接传入 `model.generate()`。

## 输出类

### GenerateOutput

由 `model.generate()` 返回。

| 属性 | 类型 | 说明 |
|---|---|---|
| `text` | `str` | 生成的文本（已去除思考标签）。 |
| `thinking` | `str \| None` | 模型的推理内容（从 `<think>` 标签中提取），或 `None`。 |
| `profile` | [`ProfileData`](#profiledata) | 性能指标。 |

### ProfileData

生成调用的性能分析数据。

| 属性 | 类型 | 说明 |
|---|---|---|
| `ttft` | `int` | 首 token 时间（毫秒）。 |
| `prompt_time` | `int` | 提示处理时间（毫秒）。 |
| `decode_time` | `int` | 解码时间（毫秒）。 |
| `prompt_tokens` | `int` | 提示 token 数。 |
| `generated_tokens` | `int` | 生成的 token 数。 |
| `prefill_speed` | `float` | 预填充速度（tokens/s）。 |
| `decode_speed` | `float` | 解码速度（tokens/s）。 |
| `stop_reason` | `str \| None` | 生成停止原因（如 `"eos"`、`"limit"`）。 |

### TextIteratorStreamer

由 `model.generate(..., stream=True)` 返回。在生成时逐块返回解码文本。

```python
streamer = model.generate(prompt, max_new_tokens=256, stream=True)
for chunk in streamer:
    print(chunk, end="", flush=True)

# 迭代结束后可获取最终输出：
final = streamer.output  # GenerateOutput
```

| 方法/属性 | 说明 |
|---|---|
| `__iter__()` | 在生成时逐块返回 `str` 文本。 |
| `output` | `GenerateOutput \| None`——迭代完成后可用的完整结果。 |
| `cancel()` | 在下一个 token 边界停止生成。 |

## 模型管理器

CLI 使用的模型管理器可通过 `geniex.model_manager` 以代码方式调用。

```python
from geniex import model_manager as mm

mm.pull("qwen3")                       # 通过别名或 "org/repo[:quant]" 下载
models = mm.list_models()              # 列出已缓存的模型名称
paths  = mm.get_paths("qwen3")        # 解析为本地文件路径
mtype  = mm.get_type("qwen3")         # "llm" 或 "vlm"
name   = mm.resolve_alias("qwen3")    # "Qualcomm/Qwen3-4B-Instruct"
mm.remove("qwen3")                    # 删除一个缓存模型
mm.clean()                            # 删除所有缓存模型
```

### `pull()`

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `model_name` | `str` | *必填* | `org/repo` 或简称别名。 |
| `quant` | `str \| None` | `None` | 量化提示（如 `"Q4_K_M"`）。 |
| `hub` | `str` | `"auto"` | `"auto"` \| `"hf"` \| `"localfs"`。 |
| `local_path` | `str \| None` | `None` | 源目录（`hub="localfs"` 时必填）。 |
| `hf_token` | `str \| None` | `None` | HuggingFace 令牌。 |
| `on_progress` | `Callable \| None` | `None` | 回调 `(files: list[FileProgress]) -> bool`；返回 `False` 取消下载。 |

### `list_models()` -> `list[str]`

返回已缓存的模型名称。

### `get_paths(model_name)` -> `ModelPaths`

将模型名称解析为本地绝对路径。

| 属性 | 类型 | 说明 |
|---|---|---|
| `model_path` | `str` | 主模型文件路径。 |
| `model_dir` | `str` | 模型所在目录。 |
| `model_name` | `str` | 规范模型名称。 |
| `plugin_id` | `str` | 插件标识符。 |
| `mmproj_path` | `str \| None` | 多模态投影器路径（仅 VLM）。 |
| `tokenizer_path` | `str \| None` | 分词器文件路径。 |
| `device_id` | `str \| None` | 设备标识符。 |

### `get_type(model_name)` -> `str`

返回 `"llm"` 或 `"vlm"`。

### `resolve_alias(alias)` -> `str`

将简称别名（如 `"qwen3"`）解析为规范 `org/repo`。

### `remove(model_name)` -> `None`

从磁盘删除缓存的模型。

### `clean()` -> `int`

删除所有缓存模型。返回删除数量。

## 运行时函数

管理 GenieX 运行时的顶层函数。

| 函数 | 说明 |
|---|---|
| `geniex.init()` | 初始化 GenieX 运行时。首次加载模型时自动调用。 |
| `geniex.deinit()` | 关闭运行时并释放资源。 |
| `geniex.version()` | 返回 SDK 版本字符串。 |
| `geniex.get_plugin_list()` | 返回可用插件 ID 的 `list[str]`（如 `["llama_cpp"]`）。 |
| `geniex.get_device_list(plugin_id)` | 返回插件的 `list[tuple[str, str]]` 设备对 `(device_id, device_name)`。 |
