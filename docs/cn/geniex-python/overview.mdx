## **GenieX Python SDK**

GenieX AI 提供完整的 Python API，支持大语言模型（LLM）和视觉语言模型（VLM）的设备端 AI 推理。其设计类似 Hugging Face `transformers` 库，为开发者提供熟悉且直观的使用体验。
