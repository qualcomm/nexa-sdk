## **支持的模型列表**
我们精选了一批高质量的 QAIRT `bin` 格式顶尖模型。

| 模型 ID | 来源 |
|---|---|
| qualcomm/Qwen3-4B | Qualcomm AI Hub |
| qualcomm/Qwen3-4B-Instruct-2507 | Qualcomm AI Hub |
| qualcomm/Qwen2.5-VL-7B-Instruct | Qualcomm AI Hub |
| llama_v3_1_8b_instruct | 自行转换并上传到个人 Hugging Face |
| llama_v3_1_sea_lion_3_5_8b_r | 自行转换并上传到个人 Hugging Face |
| llama_v3_2_1b_instruct | 自行转换并上传到个人 Hugging Face |
| llama_v3_2_3b_instruct | 自行转换并上传到个人 Hugging Face |
| llama_v3_8b_instruct | 自行转换并上传到个人 Hugging Face |
| llama_v3_elyza_jp_8b | 自行转换并上传到个人 Hugging Face |
| llama_v3_taide_8b_chat | 自行转换并上传到个人 Hugging Face |
| falcon_v3_7b_instruct | 自行转换并上传到个人 Hugging Face |

运行 `qualcomm` Hugging Face 上的模型，以 `Qwen3-4B` 为例：

```powershell
geniex infer qualcomm/Qwen3-4B
```

对于自行转换并上传到个人 Hugging Face 的模型，以 `llama_v3_2_1b_instruct` 为例，需要先将 HF 权重下载到本地，然后使用本地路径运行。

请先参考 [Hugging Face CLI 下载教程](https://huggingface.co/docs/huggingface_hub/en/guides/cli) 设置 HF CLI，然后在 Windows 笔记本上运行：

```powershell
# Set model name (must match the HF subfolder name exactly)
$model = "llama_v3_2_1b_instruct"

# Download the model from Huggingface
hf download yichqian/geniex-qairt-models `
  --local-dir $model `
  --include "$model/**"

# Pull using the downloaded folder (absolute path)
geniex pull local/$model --model-hub localfs `
  --local-path (Resolve-Path "$model\$model").Path

# Run inference
geniex infer local/$model
```
