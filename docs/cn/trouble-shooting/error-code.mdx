Nexa SDK 使用结构化错误码帮助用户在模型推理或系统运行中快速定位并解决问题。\
使用本页可快速查询任意返回错误码的含义。
##  **常见错误（100xxx）**
与无效输入、内存分配或不支持操作相关的通用错误。
| 错误码 | 名称 | 描述 |
|------------|------|-------------|
| `-100000` | `ML_ERROR_COMMON_UNKNOWN` | 未知错误 |
| `-100001` | `ML_ERROR_COMMON_INVALID_INPUT` | 无效的输入参数或句柄 |
| `-100003` | `ML_ERROR_COMMON_MEMORY_ALLOCATION` | 内存分配失败 |
| `-100004` | `ML_ERROR_COMMON_FILE_NOT_FOUND` | 文件未找到或不可访问 |
| `-100007` | `ML_ERROR_COMMON_NOT_INITIALIZED` | 库未初始化 |
| `-100013` | `ML_ERROR_COMMON_NOT_SUPPORTED` | 操作不受支持 |
| `-100201` | `ML_ERROR_COMMON_MODEL_LOAD` | 模型加载失败 |
| `-100203` | `ML_ERROR_COMMON_MODEL_INVALID` | 模型格式无效 |
| `-100301` | `ML_ERROR_COMMON_EMBEDDING_GENERATION` | 向量嵌入生成失败 |
| `-100302` | `ML_ERROR_COMMON_EMBEDDING_DIMENSION` | 嵌入维度无效 |
| `-100401` | `ML_ERROR_COMMON_RERANK_FAILED` | 重排失败 |
| `-100402` | `ML_ERROR_COMMON_RERANK_INPUT` | 重排输入无效 |
| `-100501` | `ML_ERROR_COMMON_IMG_GENERATION` | 图像生成失败 |
| `-100502` | `ML_ERROR_COMMON_IMG_PROMPT` | 图像提示词无效 |
| `-100503` | `ML_ERROR_COMMON_IMG_DIMENSION` | 图像尺寸无效 |
## **LLM 错误（200xxx）**
在语言模型分词或文本生成过程中出现的错误。
| 错误码 | 名称 | 描述 |
|------------|------|-------------|
| `-200001` | `ML_ERROR_LLM_TOKENIZATION_FAILED` | 分词失败 |
| `-200004` | `ML_ERROR_LLM_TOKENIZATION_CONTEXT_LENGTH` | 上下文长度超限 |
| `-200101` | `ML_ERROR_LLM_GENERATION_FAILED` | 文本生成失败 |
| `-200103` | `ML_ERROR_LLM_GENERATION_PROMPT_TOO_LONG` | 输入提示词过长 |
## **VLM 错误（300xxx）**
与视觉语言模型处理图像与音频相关的错误。
| 错误码 | 名称 | 描述 |
|------------|------|-------------|
| `-300001` | `ML_ERROR_VLM_IMAGE_LOAD` | 图像加载失败 |
| `-300002` | `ML_ERROR_VLM_IMAGE_FORMAT` | 不支持的图像格式 |
| `-300101` | `ML_ERROR_VLM_AUDIO_LOAD` | 音频加载失败 |
| `-300102` | `ML_ERROR_VLM_AUDIO_FORMAT` | 不支持的音频格式 |
| `-300201` | `ML_ERROR_VLM_GENERATION_FAILED` | 多模态生成失败 |
## **OCR 错误（400xxx）**
与图像中文本检测与识别相关的问题。
| 错误码 | 名称 | 描述 |
|------------|------|-------------|
| `-400001` | `ML_ERROR_OCR_DETECTION` | OCR 文本检测失败 |
| `-400002` | `ML_ERROR_OCR_RECOGNITION` | OCR 文本识别失败 |
| `-400003` | `ML_ERROR_OCR_MODEL` | OCR 模型错误 |
## **ASR 错误（500xxx）**
在自动语音识别中出现的错误。
| 错误码 | 名称 | 描述 |
|------------|------|-------------|
| `-500001` | `ML_ERROR_ASR_TRANSCRIPTION` | ASR 转写失败 |
| `-500002` | `ML_ERROR_ASR_AUDIO_FORMAT` | 不支持的 ASR 音频格式 |
| `-500003` | `ML_ERROR_ASR_LANGUAGE` | 不支持的 ASR 语言 |
## **TTS 错误（600xxx）**
与语音合成、音色选择或音频输出格式相关的问题。
| 错误码 | 名称 | 描述 |
|------------|------|-------------|
| `-600001` | `ML_ERROR_TTS_SYNTHESIS` | TTS 合成失败 |
| `-600002` | `ML_ERROR_TTS_VOICE` | TTS 音色未找到 |
| `-600003` | `ML_ERROR_TTS_AUDIO_FORMAT` | TTS 音频格式错误 |
