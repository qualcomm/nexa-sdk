GenieX 是为高通平台打造的端侧生成式 AI 推理运行时。它是 Qualcomm GENIE 的社区版，让你用几行代码即可在 Hexagon NPU、Adreno GPU 或 CPU 上本地运行前沿语言模型与视觉语言模型。

## **架构**

<img
  src="/Mintlify-image/geniex_arch.png"

GenieX 通过同一套 SDK 暴露五种入口——CLI、Python、Java/Kotlin、Docker 以及兼容 OpenAI 协议的服务器，并按需分发到 **llama.cpp 插件**（覆盖 CPU / GPU / Hexagon HTP 的 GGML 内核）或 **QAIRT 插件**（高通 AI Runtime，仅 NPU）。同一套 SDK 可运行在 Windows ARM64、Android 与 Linux ARM64 上。

## **GenieX 能做什么**

- **本地运行模型** —— 覆盖骁龙 X（Windows ARM64）、骁龙 8 至尊版（Android）以及 Dragonwing IoT 设备。
- **选择后端** —— `llama.cpp` 适配任意社区 GGUF 模型，`QAIRT` 适配 AI Hub 预编译的 NPU 模型包。
- **构建应用** —— 通过 CLI、兼容 OpenAI 协议的本地服务器、Python SDK、Android SDK 或 Docker 镜像。
