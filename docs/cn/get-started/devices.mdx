GenieX 覆盖三类骁龙硬件家族，每一类都对应不同的接入方式与后端策略。

## **计算（Windows ARM64）**

骁龙 X 系列笔记本与台式机——Copilot+ PC。

**支持的芯片组**
- 骁龙 X Elite
- 骁龙 X2 Elite

**如何获取**
- **本地** —— 任意骁龙 X 系列笔记本或台式机。
- **云端** —— 在 [Qualcomm Developer Cloud — Compute 页面](https://qdc.qualcomm.com/devices/compute) 启动 `Snapdragon X Elite` 或 `Snapdragon X2 Elite` 设备。

**接入方式** —— [CLI](/cn/run/cli/quickstart)、[Python SDK](/cn/run/python/quickstart)、[本地服务器](/cn/run/cli/local-server)。

## **移动（Android）**

骁龙 8 系列移动平台。

**支持的芯片组**
- 骁龙 8 至尊版（`SM8750`）
- 骁龙 8 至尊版 Gen 5（`SM8850`）
- 骁龙 8 至尊版 Gen 6

**如何获取**
- **本地** —— 任意骁龙 8 至尊版系列 Android 手机，开启 USB 调试。
- **云端** —— 在 [Qualcomm Developer Cloud — Mobile 页面](https://qdc.qualcomm.com/devices/mobile) 启动 `Snapdragon 8 Elite` 或 `Snapdragon 8 Elite Gen 5` 设备。

**接入方式** —— [Android SDK](/cn/run/android/quickstart)（Kotlin，Maven Central）。

## **IoT（Linux ARM64）**

面向边缘推理的高通 IoT 平台——智能摄像头、机器人、工业自动化等。

**支持的芯片组**
- 高通 QCS9075
- 高通 QCS9100
- 高通 QCS8275

**如何获取**
- **本地** —— 高通 IoT 开发套件（如 Dragonwing RB3 Gen 2、QCS8550 HDK），运行 Linux ARM64。
- **云端** —— 在 [Qualcomm Developer Cloud — IoT 页面](https://qdc.qualcomm.com/devices/iot) 启动 `Qualcomm Dragonwing IQ-9075` 或 `Qualcomm Dragonwing IQ-9100 Beta` 设备。

**接入方式** —— [原生安装](/cn/run/cli/install)或 [Docker 镜像](/cn/run/cli/install)，Linux ARM64 带 NPU 加速。
