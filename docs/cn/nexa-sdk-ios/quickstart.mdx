## **安装**

您可以从官方分发链接直接下载适用于 iOS/macOS 的 [NexaSdk.xcframework](https://nexa-model-hub-bucket.s3.us-west-1.amazonaws.com/public/ios/latest/NexaSdk.xcframework.zip)

### **添加依赖**
将下载好的 XCFramework 解压后 后拖入 Xcode 项目即可。

## **运行你的第一个模型**

<Steps>

  <Step title="加载模型">
    创建 ASR 实例并加载模型：

    ```swift
    import Foundation
    import NexaSdk

    let asr = try Asr(plugin: .ane)
    try await asr.load(from: "<path/to/model/dir>")
    ```
  </Step>

  <Step title="转写音频文件">
    使用已加载的模型转写音频：

    ```swift
    let asrResponse = try await asr.transcribe(options: .init(audioPath: "<path/to/audio>"))
    print(asrResponse.asrResult.transcript)
    ```
  </Step>
</Steps>

<Note>
  **从 HuggingFace 加载模型**
  
  可以通过官方 [Hub](https://github.com/huggingface/swift-transformers) 库下载模型，然后使用 `load(from:)` 加载：
  
  ```swift
  import Hub
  import NexaSdk

  let modelDir = try await Hub.snapshot(from: "NexaAI/parakeet-tdt-0.6b-v3-ane") { progress in
    print(progress.fractionCompleted)
  }
  let asr = try Asr(plugin: .ane)
  try await asr.load(from: modelDir)
  ```
</Note>
