## **插件与设备选择**

### 插件

通过 `plugin_id` 选择推理后端：

```kotlin
val plugin_id: String?   // "llama_cpp" | "qairt" | null
```

| `plugin_id`   | 后端                             | 模型格式                 | 设备                           |
|---------------|----------------------------------|--------------------------|--------------------------------|
| `"llama_cpp"` | llama.cpp + GGML Hexagon 后端    | GGUF                     | CPU / Adreno GPU / Hexagon NPU |
| `"qairt"`     | Qualcomm AI Runtime              | AI Hub 预编译 bin        | 仅 Hexagon NPU                 |
| `null`        | SDK 根据模型路径自动选择         | —                        | —                              |

常量通过 [`PluginIdValue`](https://github.com/qcom-ai-hub/geniex/blob/main/bindings/android/app/src/main/java/com/geniex/sdk/bean/InputPluginBase.kt) 暴露（`LLAMA_CPP`、`QAIRT`）。

### 设备

硬件别名，由 JNI 层转发给原生 SDK 中的 `geniex_resolve_device`。

```kotlin
val device_id: String?   // "cpu" | "gpu" | "npu" | null
```

| 别名    | 效果                                                        |
|---------|-------------------------------------------------------------|
| `null`  | 插件默认值 — `llama_cpp` 为 `npu`，`qairt` 为 `npu`。      |
| `"npu"` | Hexagon NPU 加速。骁龙推荐。                                |
| `"gpu"` | Adreno GPU via OpenCL（仅 `llama_cpp`）。                   |
| `"cpu"` | 纯 CPU。强制 `nGpuLayers = 0`。                             |

<Note>
QAIRT 仅支持 NPU。传入 `"cpu"` 或 `"gpu"` 时会记录警告并回退到 NPU，不会报错。
</Note>

## **模型管理器**

模型通过内置的 Rust 模型管理器在设备上拉取。请勿手动 `adb push` 模型权重 — 使用 `ModelManagerWrapper`。

### `ModelManagerWrapper`

```kotlin
// 初始化（幂等 — 可在每次 Activity.onCreate 调用）。
GeniexSdk.getInstance().init(context)

// 拉取模型，流式进度。
ModelManagerWrapper.pullFlow(
    ModelPullInput(
        model_name = "unsloth/Qwen3-0.6B-GGUF",
        quant      = "Q4_0",
        hub        = HubSource.HUGGINGFACE,
    )
).collect { event ->
    when (event) {
        is ModelManagerWrapper.PullEvent.Progress  -> /* 更新 UI */
        ModelManagerWrapper.PullEvent.Completed    -> /* 完成 */
        is ModelManagerWrapper.PullEvent.Error     -> /* 显示错误 */
    }
}

// 解析已拉取模型的磁盘路径。
val paths: ModelPaths? = ModelManagerWrapper.getPaths("unsloth/Qwen3-0.6B-GGUF")

// 清单 / 清理。
ModelManagerWrapper.list()              // List<String>
ModelManagerWrapper.remove("org/repo") // 0 = 成功
ModelManagerWrapper.clean()            // 清除所有缓存模型
```

### `ModelPullInput`

```kotlin
data class ModelPullInput(
    val model_name:   String,                     // "org/repo" 或别名
    val quant:        String? = null,             // 如 "Q4_0"、"Q4_K_M"
    val hub:          HubSource = HubSource.AUTO, // AUTO 根据 model_name 路由
    val local_path:   String? = null,             // 仅当 hub == LOCALFS
    val hf_token:     String? = null,             // 回退到 GENIEX_HFTOKEN 环境变量
    val chipset:      String? = null,             // Android 上 AI Hub 必填（如 "SM8750"）
    val display_name: String? = null,
)
```

### `HubSource`

```kotlin
enum class HubSource(val value: Int) {
    AUTO(0),          // 根据前缀路由（如 qualcomm/* → AIHUB）
    HUGGINGFACE(1),
    MODELSCOPE(2),
    AIHUB(3),
    VOLCES(4),
    LOCALFS(127),
}
```

### `ModelPaths`

由 `getPaths()` 返回。直接传入 `LlmCreateInput` / `VlmCreateInput`：

```kotlin
data class ModelPaths(
    val model_path:     String,
    val model_dir:      String,
    val model_name:     String,
    val plugin_id:      String,            // 权威值 — 优先于 UI 选择
    val mmproj_path:    String? = null,    // VLM 视觉投影权重
    val tokenizer_path: String? = null,
    val device_id:      String? = null,
)
```

<Note>
Android 上 AI Hub 拉取**必须**提供 `chipset`。Rust 端仅在 Windows-on-Snapdragon 上自动检测。骁龙 8 Elite 使用 `"SM8750"`，骁龙 8 Elite Gen 5 使用 `"SM8850"`。
</Note>

## **数据结构**

### `LlmCreateInput`

```kotlin
data class LlmCreateInput(
    val model_name:     String,
    val model_path:     String,
    val tokenizer_path: String? = null,
    val config:         ModelConfig,
    val plugin_id:      String? = null,
    val device_id:      String? = null,
)
```

### `VlmCreateInput`

```kotlin
data class VlmCreateInput(
    val model_name:  String,
    val model_path:  String,
    val mmproj_path: String? = null,     // 视觉投影权重（GGUF VLM 用）
    val config:      ModelConfig,
    val plugin_id:   String? = null,
    val device_id:   String? = null,
)
```

### `ModelConfig`

```kotlin
data class ModelConfig(
    var nCtx:                  Int     = 2048,     // 上下文大小；0 = 模型默认值
    var nThreads:              Int     = 8,
    var nThreadsBatch:         Int     = 8,
    var nBatch:                Int     = 2048,
    var nUBatch:               Int     = 512,
    var nSeqMax:               Int     = 1,
    var nGpuLayers:            Int     = 0,
    val chat_template_path:    String  = "",
    val chat_template_content: String  = "",
    val max_tokens:            Int     = 2048,
    val enable_thinking:       Boolean = false,
    val verbose:               Boolean = false,
)
```

<Note>
JNI 根据 `device_id` 重写 `nGpuLayers`：`cpu` 强制为 0，`npu` 强制为 999。`gpu` 使用传入值 — 设置 999 可卸载所有层。
</Note>

### `ChatMessage`

```kotlin
data class ChatMessage(
    var role:    String,   // "system" | "user" | "assistant"
    var content: String,
)
```

### `VlmChatMessage` / `VlmContent`

```kotlin
data class VlmChatMessage(
    val role:     String?,                 // "system" | "user" | "assistant"
    val contents: List<VlmContent>,
)

data class VlmContent(
    val type: String?,                     // "text" | "image"
    val text: String?,                     // 文本内容，或图像的绝对文件路径
)
```

### `GenerationConfig`

```kotlin
data class GenerationConfig(
    var maxTokens:     Int              = 32,
    var stopWords:     Array<String>?   = null,
    var stopCount:     Int              = 0,
    var nPast:         Int              = 0,
    var samplerConfig: SamplerConfig?   = null,
    var imagePaths:    Array<String>?   = null,
    var imageCount:    Int              = 0,
    var audioPaths:    Array<String>?   = null,
    var audioCount:    Int              = 0,
)
```

<Note>
默认 `maxTokens` 为 **32**。大多数场景应设置更高的值（如 `maxTokens = 2048`）。
</Note>

### `LlmStreamResult`

```kotlin
sealed class LlmStreamResult {
    data class Token(val text: String)              : LlmStreamResult()
    data class Completed(val profile: ProfilingData) : LlmStreamResult()
    data class Error(val throwable: Throwable)      : LlmStreamResult()
}
```

## **llama.cpp（GGUF 模型）**

在 CPU、Adreno GPU 或 Hexagon NPU 上运行任意 GGUF 模型。通过 `device_id` 控制设备选择。

### LLM

```kotlin
val paths = ModelManagerWrapper.getPaths("unsloth/Qwen3-0.6B-GGUF")
    ?: error("模型未下载")

LlmWrapper.builder()
    .llmCreateInput(
        LlmCreateInput(
            model_name = paths.model_name,
            model_path = paths.model_path,
            config     = ModelConfig(nCtx = 4096),
            plugin_id  = "llama_cpp",
            device_id  = null,       // null → npu（骁龙推荐）
        )
    )
    .build()
    .onSuccess { llmWrapper = it }
    .onFailure { println("错误: ${it.message}") }

val chat = arrayListOf(ChatMessage("user", "什么是人工智能？"))

llmWrapper.applyChatTemplate(chat.toTypedArray(), null, false).onSuccess { t ->
    llmWrapper.generateStreamFlow(t.formattedText, GenerationConfig(maxTokens = 2048)).collect { result ->
        when (result) {
            is LlmStreamResult.Token     -> print(result.text)
            is LlmStreamResult.Completed -> println("\n完成")
            is LlmStreamResult.Error     -> println("错误: ${result.throwable}")
        }
    }
}
```

#### 设备变体

| 目标 | `device_id` | 说明 |
|---|---|---|
| 骁龙 NPU（推荐） | `"npu"` 或 `null` | Hexagon NPU 加速。 |
| Adreno GPU (OpenCL) | `"gpu"` | 设置 `nGpuLayers = 999` 卸载所有层。 |
| 纯 CPU | `"cpu"` | 适用于任何 ARM64 设备。 |

### VLM

GGUF VLM 需要两个产物：LLM 权重（`model_path`）和视觉投影（`mmproj_path`）。两者都来自 `getPaths()`：

```kotlin
val paths = ModelManagerWrapper.getPaths("unsloth/Qwen3-VL-2B-Instruct-GGUF")
    ?: error("模型未下载")

VlmWrapper.builder()
    .vlmCreateInput(
        VlmCreateInput(
            model_name  = paths.model_name,
            model_path  = paths.model_path,
            mmproj_path = paths.mmproj_path,
            config      = ModelConfig(nCtx = 4096),
            plugin_id   = "llama_cpp",
            device_id   = null,
        )
    )
    .build()
    .onSuccess { vlmWrapper = it }

val msg = VlmChatMessage(
    role     = "user",
    contents = listOf(
        VlmContent("image", "/storage/emulated/0/Pictures/example.jpg"),
        VlmContent("text",  "描述这张图片。"),
    ),
)
val chat = arrayListOf(msg)

vlmWrapper.applyChatTemplate(chat.toTypedArray(), null, false).onSuccess { t ->
    val gen = vlmWrapper.injectMediaPathsToConfig(chat.toTypedArray(), GenerationConfig(maxTokens = 2048))
    vlmWrapper.generateStreamFlow(t.formattedText, gen).collect { result ->
        when (result) {
            is LlmStreamResult.Token     -> print(result.text)
            is LlmStreamResult.Completed -> println("\n完成")
            is LlmStreamResult.Error     -> println("错误: ${result.throwable}")
        }
    }
}
```

<Note>
始终将 `t.formattedText`（经过聊天模板处理的提示）传入 `generateStreamFlow`，**不要**传入原始用户文本。原生管线将提示视为已模板化。
</Note>

## **QAIRT（AI Hub NPU 模型）**

来自 Qualcomm AI Hub 的预编译模型。仅 NPU，固定到特定芯片组（`SM8750` = 骁龙 8 Elite，`SM8850` = 骁龙 8 Elite Gen 5）。

### 下载 AI Hub 模型

```kotlin
ModelManagerWrapper.pullFlow(
    ModelPullInput(
        model_name = "qualcomm/Qwen2.5-VL-7B-Instruct",
        hub        = HubSource.AUTO,     // AUTO 将 `qualcomm/*` 路由到 AI Hub
        chipset    = "SM8750",           // Android 上必填
    )
).collect { event ->
    when (event) {
        is ModelManagerWrapper.PullEvent.Progress  -> updateProgressBar(event.files)
        ModelManagerWrapper.PullEvent.Completed    -> println("完成")
        is ModelManagerWrapper.PullEvent.Error     -> println("错误 ${event.code}: ${event.message}")
    }
}
```

### 支持的模型

| 模态 | Hub 仓库 |
|---|---|
| LLM | `qualcomm/Qwen3-4B-Instruct-2507` |
| VLM | `qualcomm/Qwen2.5-VL-7B-Instruct` |

### LLM

```kotlin
val paths = ModelManagerWrapper.getPaths("qualcomm/Qwen3-4B-Instruct-2507")
    ?: error("模型未下载")

LlmWrapper.builder()
    .llmCreateInput(
        LlmCreateInput(
            model_name = paths.model_name,
            model_path = paths.model_path,
            config     = ModelConfig(max_tokens = 2048, enable_thinking = false),
            plugin_id  = "qairt",
            device_id  = null,           // null → NPU（QAIRT 唯一选项）
        )
    )
    .build()
    .onSuccess { llmWrapper = it }
    .onFailure { println("错误: ${it.message}") }

val chat = arrayListOf(ChatMessage("user", "什么是人工智能？"))

llmWrapper.applyChatTemplate(chat.toTypedArray(), null, false).onSuccess { t ->
    llmWrapper.generateStreamFlow(t.formattedText, GenerationConfig()).collect { result ->
        when (result) {
            is LlmStreamResult.Token     -> print(result.text)
            is LlmStreamResult.Completed -> println("\n完成")
            is LlmStreamResult.Error     -> println("错误: ${result.throwable}")
        }
    }
}
```

<Note>
QAIRT 会拒绝 `nGpuLayers != 0` 和 `nCtx != 0`（`PARAM_NOT_SUPPORTED`）— KV 缓存和上下文长度在 AI Hub 编译时已固定。保持默认值，仅使用 `max_tokens` / `enable_thinking`。
</Note>

### VLM

```kotlin
val paths = ModelManagerWrapper.getPaths("qualcomm/Qwen2.5-VL-7B-Instruct")
    ?: error("模型未下载")

VlmWrapper.builder()
    .vlmCreateInput(
        VlmCreateInput(
            model_name  = paths.model_name,
            model_path  = paths.model_path,
            mmproj_path = paths.mmproj_path,
            config      = ModelConfig(max_tokens = 2048, enable_thinking = false),
            plugin_id   = "qairt",
            device_id   = null,
        )
    )
    .build()
    .onSuccess { vlmWrapper = it }

val msg = VlmChatMessage(
    role     = "user",
    contents = listOf(
        VlmContent("image", "/storage/emulated/0/Pictures/cat.jpg"),
        VlmContent("text",  "这张图片里有什么？"),
    ),
)
val chat = arrayListOf(msg)

vlmWrapper.applyChatTemplate(chat.toTypedArray(), null, false).onSuccess { t ->
    val gen = vlmWrapper.injectMediaPathsToConfig(chat.toTypedArray(), GenerationConfig(maxTokens = 2048))
    vlmWrapper.generateStreamFlow(t.formattedText, gen).collect { result ->
        when (result) {
            is LlmStreamResult.Token     -> print(result.text)
            is LlmStreamResult.Completed -> println("\n完成")
            is LlmStreamResult.Error     -> println("错误: ${result.throwable}")
        }
    }
}
```

<Note>
将**经过聊天模板处理的**提示（`t.formattedText`）传入 `generateStreamFlow`，不要传原始用户文本。QAIRT VLM 将提示视为已模板化 — 原始文本会产生退化输出。
</Note>
